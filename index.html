HTML

HTML ist eine Sprache für Webseiten, die den Inhalt strukturiert und angibt, wie er im Browser dargestellt wird.

Hypertext Markup Language

Im folgenden sehen wir uns an, wie du deiner Website Struktur verleihen kannst!

CSS

CSS ist eine Sprache, die verwendet wird, um das Aussehen und das Layout von Webseiten zu gestalten, indem sie Stile wie Farben, Schriftarten und Abstände festlegt.

Cascading Style Sheets

Damit deine Website auch gut aussieht, lernen wir nun, wie wir ihr Aussehen verändern.

JS

JS macht Webseiten interaktiv, indem es Funktionen wie Inhaltsänderungen und Benutzerinteraktionen ermöglicht.

Javascript

Mit Javascript können wir der Website zum Schluss noch ein wenig funktionalität verleihen.
